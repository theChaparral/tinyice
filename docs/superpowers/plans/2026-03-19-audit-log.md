# Audit Log Implementation Plan

> **For agentic workers:** Use superpowers:subagent-driven-development to implement this plan.

**Goal:** Add structured audit logging for all admin actions with SQLite storage, paginated API, and admin UI.

**Architecture:** AuditEntry GORM model in existing SQLite DB. Server.Audit() helper called from handlers. GET /api/security/audit with pagination. Security page gets Audit Log tab.

**Tech Stack:** Go/GORM/SQLite, Preact/Signals frontend
